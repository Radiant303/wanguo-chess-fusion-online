特殊融合规则

  • 兵 + 兵 = 炮
    ▼
    陈胜、吴广领导秦末农民起义。
    公元前209年，陈胜、吴广等戍边士卒在大泽乡因大雨延误行期，被迫揭竿而起，发动了中国历史上第一次大规模农民起义，沉重打击了秦王朝的统治根基。
  • 兵 + 马 = 车
    ▼
    卫青与霍去病北击匈奴。
    汉武帝时期，卫青、霍去病先后率军出击匈奴，卫青收复河朔之地，霍去病斩获匈奴主力、封狼居胥，二人联手重创匈奴势力，打通河西走廊，巩固了汉朝北方疆域。
  • 兵 + 炮 = 铳 (隔1或2子吃)
    ▼
    义和团与清军部分力量联合抗洋。
    19世纪末，义和团运动兴起，以“扶清灭洋”为口号反抗列强侵略，后期与部分清军形成临时同盟，在天津、北京等地与八国联军展开激战，是近代中国人民反抗外来侵略的重要组成部分。
    升级版的炮（铳）可以隔着一个子吃，也可以隔着两个子吃。
  • 马 + 车 = 骠 (兼具马车走法，会蹩腿)
    ▼
    薛仁贵与唐军主力征西。
    唐朝初年，薛仁贵追随唐军出征西域，凭借骁勇善战立下赫赫战功，先后参与平定西突厥、征伐高句丽等战事，“三箭定天山”便是其经典战绩，为唐朝稳定西域局势作出重要贡献。
    升级版的马（骠）兼具马和车的走法，但会蹩腿。
  • 马 + 象 = 骏 (兼具马象走法，会蹩腿，能过河)
    ▼
    岳飞与韩世忠联手抗金。
    南宋时期，金兵南下入侵，岳飞率领岳家军、韩世忠统领水军，分别在郾城大捷、黄天荡之战等战役中重创金军，二人同为南宋抗金名将，坚守江淮防线，力保南宋半壁江山。
    升级版的马（骏）兼具马和象的走法，但会蹩腿，且能过河。
  • 车 + 炮 = 軳 (水平远程打击，无上下)
    ▼
    辽沈战役中东北野战军主力与炮兵协同作战。
    1948年，东北野战军发起辽沈战役，主力部队与配属炮兵部队密切配合，先后攻克锦州、长春、沈阳等重镇，彻底解放东北全境，为全国解放战争的胜利奠定基础。
    升级版的车（軳）水平线上可以隔着任意子吃，但失去上下移动能力。
  • 车 + 兵 = 轀 (兵走法能动两格，移两次)
    ▼
    刘邓大军千里跃进大别山。
    1947年，刘伯承、邓小平率领晋冀鲁豫野战军主力，突破国民党军黄河防线，千里跃进大别山，深入中原腹地，揭开了中国人民解放军战略反攻的序幕。
    升级版的兵（轀）每次最多可以移动两格，可以移动两次(必须移动两次)。
  • 士 + 士 = 仕 (全盘斜走一格，锁定己方将帅)
    ▼
    1936年张学良、杨虎城发动西安事变。
    为逼迫蒋介石停止内战、联共抗日，张学良、杨虎城在西安扣押蒋介石，发动“兵谏”，后经多方斡旋事变和平解决，成为扭转时局的关键，推动了抗日民族统一战线的初步形成。
    升级版的士（仕）可以出九宫格斜着走，但是己方存在仕时，己方将帅失去移动能力。
